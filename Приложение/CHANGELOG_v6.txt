✨ НОВЫЕ ФИЧИ

1. УВЕДОМЛЕНИЯ О КРИТИЧЕСКИХ ТОВАРАХ
   Файл: MainForm.cs
   Метод: CheckCriticalStock()
   Описание:
   - Вызывается при загрузке MainForm
   - Проверяет товары без запаса (количество = 0)
   - Проверяет товары с низким запасом (количество ≤ минимума)
   - Показывает всплывающее уведомление с информацией
   - Показывает до 3 товаров в уведомлении
   Преимущества: Администратор сразу видит проблемы с запасами

2. ФИЛЬТРАЦИЯ ПО СТАТУСУ ТОВАРОВ
   Файл: MainForm.cs
   Метод: ShowProducts()
   Описание:
   - Добавлен ComboBox "Статус:" с 4 вариантами:
     * Все - показывает все товары
     * ✅ В наличии - количество > минимума
     * ⚠️ Низкий запас - количество ≤ минимума и > 0
     * 🔴 Нет в наличии - количество = 0
   - Фильтр работает в реальном времени
   - Работает одновременно с поиском
   Преимущества: Быстро найти товары по статусу

3. ФИЛЬТРАЦИЯ ПО КАТЕГОРИЯМ
   Файл: MainForm.cs
   Метод: ShowProducts()
   Описание:
   - Добавлен ComboBox "Категория:" с динамическим списком категорий
   - Загружает все категории из БД
   - Фильтр работает в реальном времени
   - Работает одновременно с поиском и фильтром по статусу
   Преимущества: Удобно фильтровать товары по категориям

4. КОМБИНИРОВАННЫЙ ПОИСК И ФИЛЬТРАЦИЯ
   Файл: MainForm.cs
   Метод: ShowProducts()
   Описание:
   - Поиск + фильтр по статусу + фильтр по категориям работают вместе
   - Используется Action<string, int, int> для обновления таблицы
   - Все три фильтра срабатывают при изменении любого из них
   - Результаты обновляются мгновенно
   Преимущества: Очень мощный и гибкий поиск
