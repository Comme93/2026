   - На странице "Товары" добавлен фильтр по статусу:
     * Все - показывает все товары
     * ✅ В наличии - товары с количеством > минимума
     * ⚠️ Низкий запас - товары с количеством ≤ минимума
     * 🔴 Нет в наличии - товары с количеством = 0
   - Фильтр работает в реальном времени

   - Фильтр работает одновременно с поиском и фильтром по статусу
   - Очень удобно для быстрого поиска товаров

🔍 УЛУЧШЕННЫЙ ПОИСК
   - Поиск работает одновременно с фильтрами

🚀 КАК ИСПОЛЬЗОВАТЬ НОВЫЕ ФИЧИ

1. УВЕДОМЛЕНИЯ О КРИТИЧЕСКИХ ТОВАРАХ:
   - При входе в приложение автоматически проверяются товары
   - Если есть проблемы - показывается уведомление
   - Нажмите OK чтобы закрыть уведомление

2. ФИЛЬТРАЦИЯ ПО СТАТУСУ:
   - Перейти на вкладку "Товары"
   - Выбрать статус из выпадающего списка
   - Таблица обновится автоматически

3. ФИЛЬТРАЦИЯ ПО КАТЕГОРИЯМ:
   - Перейти на вкладку "Товары"
   - Выбрать категорию из выпадающего списка
   - Таблица обновится автоматически

4. КОМБИНИРОВАННЫЙ ПОИСК:
   - Введите текст в поле поиска
   - Выберите статус (опционально)
   - Выберите категорию (опционально)
   - Таблица обновится с учётом всех фильтров

5. ВЫХОД ИЗ АККАУНТА:
   - Нажать кнопку "🚪 Выход" в правом верхнем углу
   - Вернётесь на форму входа
   - Сессия будет очищена
   - Можно войти с другим аккаунтом
