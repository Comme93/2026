✨ ЧТО ТЕСТИРОВАТЬ

1. ВЫХОД ИЗ АККАУНТА (ИСПРАВЛЕНО!)
   ✓ Нажмите кнопку "🚪 Выход" в правом верхнем углу
   ✓ Должны вернуться на форму входа
   ✓ Попробуйте войти с другим аккаунтом
   ✓ Сессия должна быть очищена

2. УВЕДОМЛЕНИЯ О КРИТИЧЕСКИХ ТОВАРАХ
   ✓ При входе должно показаться уведомление
   ✓ Если есть товары без запаса - будет 🔴 уведомление
   ✓ Если только низкий запас - будет 🟡 уведомление
   ✓ Нажмите OK чтобы закрыть

3. ФИЛЬТРАЦИЯ ПО СТАТУСУ
   ✓ Перейдите на вкладку "📦 Товары"
   ✓ Найдите выпадающий список "Статус:"
   ✓ Выберите "✅ В наличии" - должны остаться только товары с запасом
   ✓ Выберите "⚠️ Низкий запас" - должны остаться товары с низким запасом
   ✓ Выберите "🔴 Нет в наличии" - должны остаться товары без запаса
   ✓ Выберите "Все" - должны вернуться все товары

4. ФИЛЬТРАЦИЯ ПО КАТЕГОРИЯМ
   ✓ На странице "Товары" найдите выпадающий список "Категория:"
   ✓ Выберите конкретную категорию (например, "Электроника")
   ✓ Таблица должна показать только товары этой категории
   ✓ Выберите "Все категории" - должны вернуться все товары

5. КОМБИНИРОВАННЫЙ ПОИСК
   ✓ Введите текст в поле поиска (например, "ELEC")
   ✓ Выберите статус "✅ В наличии"
   ✓ Выберите категорию "Электроника"
   ✓ Таблица должна показать только электронику в наличии с артикулом ELEC

6. ПАНЕЛЬ УПРАВЛЕНИЯ
   ✓ Нажмите на "📊 Панель управления" в левой панели
   ✓ Должны видеть статистику (всего товаров, низкий запас, стоимость, средняя цена)
   ✓ Должна быть таблица товаров с низким запасом
   ✓ Должны быть 5 кнопок быстрого доступа

7. ДОБАВЛЕНИЕ ТОВАРА (только для админа)
   ✓ Перейдите на "📦 Товары"
   ✓ Нажмите кнопку "➕ Добавить"
   ✓ Заполните форму и нажмите "Сохранить"
   ✓ Товар должен появиться в таблице

8. РЕДАКТИРОВАНИЕ ТОВАРА (только для админа)
   ✓ Выберите товар в таблице
   ✓ Нажмите "✏️ Редактировать" или двойной клик
   ✓ Измените данные и нажмите "Сохранить"
   ✓ Изменения должны отобразиться в таблице

9. УДАЛЕНИЕ ТОВАРА (только для админа)
   ✓ Выберите товар в таблице
   ✓ Нажмите "🗑️ Удалить"
   ✓ Подтвердите удаление
   ✓ Товар должен исчезнуть из таблицы

10. ВЫГРУЗКА В CSV
    ✓ На любой странице нажмите "📥 Выгрузить"
    ✓ Выберите место сохранения
    ✓ Файл должен сохраниться с названием типа: Товары_2026-05-05_12-00-00.csv
    ✓ Откройте файл в Excel или текстовом редакторе

11. РОЛЕВОЙ ДОСТУП
    ✓ Войдите как admin - должны видеть все кнопки редактирования
    ✓ Выйдите и войдите как user - кнопки редактирования должны быть скрыты
    ✓ Для админа должна быть вкладка "👥 Пользователи"
    ✓ Для user этой вкладки не должно быть
