# 🚀 Инструкция по запуску и тестированию

## Шаг 1: Подготовка

1. Убедитесь, что установлен .NET 8.0:
```bash
dotnet --version
```

2. Убедитесь, что SQL Server (LocalDB) установлен и запущен

## Шаг 2: Запуск приложения

### Вариант 1: Из Visual Studio
1. Откройте `WarehouseManagement.sln`
2. Нажмите F5 или кнопку "Run"

### Вариант 2: Из командной строки
```bash
cd C:\Users\aizhukov\Desktop\ПРОБА
dotnet run
```

### Вариант 3: Запуск скомпилированного .exe
```bash
C:\Users\aizhukov\Desktop\ПРОБА\WarehouseManagement\bin\Release\net8.0-windows\WarehouseManagement.exe
```

4. Выполните скрипт из файла `SampleData.sql`

## 🔧 Решение проблем

### Проблема: "Could not connect to database"
**Решение:** Убедитесь, что SQL Server (LocalDB) запущен:
```bash
sqllocaldb start mssqllocaldb
```

### Проблема: "Login failed"
**Решение:** Проверьте учетные данные (admin/admin)

### Проблема: Приложение не запускается
**Решение:** Убедитесь, что установлен .NET 8.0:
```bash
dotnet --version
```

## 📝 Заметки

- Все данные хранятся в SQL Server LocalDB
- База данных создается автоматически при первом запуске
- Пароль администратора хеширован (SHA256)
- Все действия логируются в таблице UserActionLogs
