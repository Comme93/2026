# 📦 Warehouse Management System

Профессиональная система управления складом с красивым минималистичным дизайном, анимациями и полным функционалом.

## ✨ Особенности

- 🎨 **Красивый современный дизайн** - темная тема с голубыми акцентами
- ⚡ **Плавные анимации** - fade-in эффекты при загрузке
- 🔐 **Система аутентификации** - безопасный вход с хешированием паролей
- 📊 **Панель управления** - статистика и аналитика в реальном времени
- 📦 **Управление товарами** - полный CRUD функционал
- 📂 **Категории** - организация товаров по категориям
- 🚚 **Поставщики** - управление поставщиками
- 📥 **Приход товаров** - учет поступления товаров
- 📤 **Расход товаров** - учет отпуска товаров
- 📋 **Логирование** - полная история действий пользователей
- 💾 **SQL Server** - надежное хранилище данных
- 🎯 **Минималистичный UI** - интуитивный интерфейс

## 🚀 Быстрый старт

### Требования
- .NET 8.0 или выше
- SQL Server (LocalDB)
- Windows 7 SP1 или выше

### Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd ПРОБА
```

2. Восстановите зависимости:
```bash
dotnet restore
```

3. Соберите проект:
```bash
dotnet build -c Release
```

4. Запустите приложение:
```bash
dotnet run
```

## 📊 Функциональность

### Панель управления
- Общая статистика по товарам
- Товары с низким запасом
- Общая стоимость инвентаря
- Количество активных категорий

### Управление товарами
- Добавление новых товаров
- Редактирование информации
- Удаление товаров
- Отслеживание количества
- Управление минимальным запасом

### Система логирования
- Полная история действий пользователей
- Отслеживание времени действий
- Детали операций

## 🔒 Безопасность

- Хеширование паролей (SHA256)
- Логирование всех действий
- Контроль доступа по ролям
- Безопасное хранение данных
