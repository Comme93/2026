1️⃣ GETTING STARTED

System Requirements:
- Windows 10 or later
- .NET 8.0 Runtime
- SQL Server LocalDB
- 4GB RAM minimum
- 500MB disk space

Installation:
1. Download WarehouseManagement.exe
2. Run the installer
3. Follow the installation wizard
4. Accept the license agreement
5. Choose installation directory
6. Complete installation
7. Launch the application

First Launch:
1. Application initializes database
2. Auto-backup service starts
3. Login form appears
4. Enter credentials (admin/admin1234 or user/user123)
5. Select language preference
6. Dashboard loads

2️⃣ LOGIN & AUTHENTICATION

Login Screen:
- Username field: Enter your login
- Password field: Enter your password
- Language selector: Choose preferred language (Russian, English, Spanish, German)
- Demo credentials: admin/admin1234 or user/user123

Two-Factor Authentication (2FA):
1. Enter username and password
2. Receive 6-digit code via email
3. Enter code in verification dialog
4. Click "Verify" button
5. Access granted

If you lose access:
- Use backup recovery codes
- Contact administrator
- Reset password (admin only)

8️⃣ TWO-FACTOR AUTHENTICATION

Enabling 2FA:
1. Click "⚙️ Settings" menu
2. Select "Security"
3. Click "Enable 2FA"
4. Choose delivery method (Email, SMS, Authenticator)
5. Enter phone number (if SMS)
6. Generate backup codes
7. Save backup codes in secure location
8. Click "Enable"

Using 2FA:
1. Enter username and password
2. Receive 6-digit code
3. Enter code in verification dialog
4. Click "Verify"
5. Access granted

Backup Codes:
- 10 one-time use codes
- Use if you lose access to 2FA device
- Each code can only be used once
- Store in secure location
- Generate new codes if needed

Disabling 2FA:
1. Click "⚙️ Settings" menu
2. Select "Security"
3. Click "Disable 2FA"
4. Confirm disabling
5. 2FA disabled

9️⃣ BACKUP & RECOVERY

Automatic Backups:
- Automatic backup every 30 minutes
- Last 10 backups retained
- Automatic cleanup of old backups
- No user action required

Manual Backup:
1. Click "💾 Backup" menu
2. Click "Create Backup" button
3. Backup starts
4. Backup completes
5. Backup added to list

Viewing Backups:
1. Click "💾 Backup" menu
2. View list of available backups
3. See backup date, size, type
4. Sort by date or size

Restoring from Backup:
1. Click "💾 Backup" menu
2. Select backup to restore
3. Click "↩️ Restore" button
4. Confirm restoration
5. ⚠️ WARNING: Current data will be overwritten
6. Restoration starts
7. Restoration completes
8. Application restarts

Deleting Backups:
1. Click "💾 Backup" menu
2. Select backup to delete
3. Click "🗑️ Delete" button
4. Confirm deletion
5. Backup deleted

🔟 TROUBLESHOOTING

Problem: Cannot login
Solution:
- Check username and password
- Verify caps lock is off
- Reset password (admin only)
- Check 2FA code expiry (10 minutes)
- Contact administrator

Problem: Application crashes
Solution:
- Restart application
- Check system requirements
- Update .NET runtime
- Check disk space
- Contact support

Problem: Database connection error
Solution:
- Check SQL Server is running
- Verify connection string
- Check firewall settings
- Restart SQL Server
- Contact IT support

Problem: Slow performance
Solution:
- Check system resources
- Close other applications
- Restart application
- Check database size
- Contact support

Problem: Backup failed
Solution:
- Check disk space
- Verify backup location
- Check file permissions
- Restart application
- Contact support

❓ FREQUENTLY ASKED QUESTIONS

Q: How often are backups created?
A: Automatic backups are created every 30 minutes. You can also create manual
   backups at any time.

Q: How many backups are retained?
A: The system keeps the last 10 backups. Older backups are automatically deleted.

Q: Can I restore a backup?
A: Yes, you can restore any backup. Warning: This will overwrite current data.

Q: What languages are supported?
A: Russian, English, Spanish, and German.

Q: How do I enable 2FA?
A: Go to Settings > Security > Enable 2FA and follow the instructions.

Q: What if I lose my 2FA device?
A: Use one of your backup recovery codes to login, then disable 2FA.

Q: Can I export reports?
A: Yes, reports can be exported to PDF, Excel, or CSV format.

Q: How do I change my password?
A: Go to Settings > Security > Change Password.

Q: Is my data encrypted?
A: Yes, sensitive data is encrypted using AES-256 encryption.

Q: How do I contact support?
A: Email: support@warehouse.local or call: +7 (999) 123-45-67
